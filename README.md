# 우아함 (우리 아이 함께)

## 📢  서비스 소개
<img src = "https://user-images.githubusercontent.com/74968390/173279625-d0c10919-3af2-488e-920a-5641e27560a8.png" width = 25%>

**우리 아이와 함께 해요!**  
부모, 자녀, 교사가 하나로 연결되어 서로의 정보를 주고 받는 서비스  
`학교 및 반 정보를 등록하여 급식표, 시간표의 정보도 얻을 수 있어요!`    

## ⚙️  기술 스택 - iOS
- **Swift**
- **Library**
  - Alamofire
  - SnapKit
